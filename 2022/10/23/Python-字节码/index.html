Py字节码

Python 的字节码是一种类似汇编指令的中间语言，一个 Python 语句会对应若干字节码指令，虚拟机一条一条执行字节码指令，从而完成程序执行。

Python 的 dis 模块支持对 Python 代码进行反汇编， 生成字节码指令。
